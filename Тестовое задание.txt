Необходимо написать авто-тест перехода с сайта Яндекс на сайт ЦИАН, далее выбрать арендное жилье по своем району. Для этого нужно написать программу, которая будет выполнять следующие действия:

1) Открытие стартовой страницы Яндекс.
2) Поиск элемента с текстом «ЦИАН» на странице Яндекс.
3) Переключение на сайт ЦИАН.
4) Нажатие кнопки «Снять» на сайте ЦИАН.
5) Поиск элемента с текстом Вашего района на поисковой странице ЦИАН.
6) Нажатие кнопки поиска на поисковой странице ЦИАН.
Для выполнения задания необходимо использовать паттерн PageObject, в тесте только вызываются методы. Залить нужно на Ваш публичный репозиторий github.com.
